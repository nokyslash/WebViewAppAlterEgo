# Changelog

## v1.1.1

### Improvements

- Updated the application launcher icon.

- External links are now opened in the device’s default web browser, improving navigation consistency and user control.

- Added gesture-based navigation:
  - Swipe left → go forward
  - Swipe right → go back

- Implemented file upload support (available on Android 5.0 and above), allowing users to attach files directly from the app.

- Added support for landscape orientation to improve usability across different device positions.

- Introduced deep linking support:
  - Forum links can now open directly in the app
  - Users can set the app as the default handler for forum URLs
